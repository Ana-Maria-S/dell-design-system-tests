Block buttons fill the width of their containers and are commonly used in product stacks, comparison tables and on smartphone breakpoints.

Primary Non-Purchase Block

Used for page actions that serve the primary purpose of the page, if unrelated to making a purchase. If the CTA is for navigation rather than performing an action, use a link styled as a button instead.


Secondary Block

Used for important page actions that are not the primary page purpose. If the CTA is for navigation rather than performing an action, use a link styled as a button instead.

Purchased Block

Used for page actions that serve the primary purpose of the page, which are related to making a purchase. If the CTA is for navigation rather than performing an action, use a link styled as a button instead.
