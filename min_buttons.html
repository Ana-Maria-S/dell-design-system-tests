Mini buttons are limited to data tables where space is at a premium. Tables may also use links (for navigation) and buttons styled as a links (for lesser actions).

Inside a container

Secondary Mini

Used for important actions within data tables that are not the primary page purpose. If the CTA is for navigation rather than performing an action, use a link styled as a button instead.

Purchased Mini

Used for actions within data tables that serve the primary purpose of the page, which are related to making a purchase. If the CTA is for navigation rather than performing an action, use a link styled as a button instead.
